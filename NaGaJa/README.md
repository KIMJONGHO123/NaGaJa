# 나가자 (NaGaJa)
**등교시간 예측 서비스** — School Departure Time Prediction Service

> 사용자의 시간표, 위치, 교통수단, 개인 준비시간을 기반으로 최적 출발 시각을 계산하고  
> IoT 물리 알람시계와 연동하여 행동을 유도하는 개인 맞춤형 등교 지원 서비스

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | Flutter (iOS / Android 크로스플랫폼) |
| 백엔드 | Firebase (Firestore, Auth, Cloud Functions, FCM) |

### UI 화면 구현
- [x] **메인 홈 화면** (`home_screen.dart`)
  - 실시간 시계 (초 단위 갱신)
  - 상태 게이지 — 여유(초록) / 나가자(노랑) / 지각위기(빨강) 색상 애니메이션
  - 준비시간·이동시간·출발시각 정보 카드
  - 준비시작 → 출발 → 도착 순서 버튼 흐름
  - 지각위기 시 택시 마지노선 카드 표시
- [x] **캘린더 화면** (`calendar_screen.dart`)
  - 주간 / 월간 뷰 전환
  - 날짜별 출결 상태 도트 시각화 (정시·지각·결석)
  - 월별 통계 (정시·지각·결석 카운트)
- [x] **설정 화면** (`settings_screen.dart`)
  - 요일별 수업 시각 설정 (TimePicker)
  - 출발지·목적지 입력
  - 교통수단 선택 (버스·지하철·도보)
  - 개인 준비시간 슬라이더 (10~90분)
  - BLE 물리 시계 연결 UI
- [x] **지각 대응 화면** (`late_response_screen.dart`)
  - 택시 마지노선 시각 대시보드
  - 마지노선 경과 시 "오늘은 포기" 메시지 자동 전환
- [x] **메인 쉘** (`main_shell.dart`) — 하단 탭 네비게이션
